# superhero-registration

## Objetivo
Cadastrar e deletar super-heróis

## Linguagens, ferramentas e dependências

- Radix-ui: `biblioteca com acessibilidade`
- React hook form
- Zod
- Phosphor react
- Sass
- Typescript

## Como iniciar a aplicação na sua máquina

- cd project
- npm i
- npm run dev
